# RampReady Deployment

## Environment Variables

Add the following to Vercel dashboard under **Settings → Environment Variables** before deploying:

```
VITE_SUPABASE_URL=your_value
VITE_SUPABASE_ANON_KEY=your_value
VITE_OPENSKY_CLIENT_ID=your_value
VITE_OPENSKY_CLIENT_SECRET=your_value
```

## Build Command

```
npm run build
```

## Output Directory

```
dist
```
